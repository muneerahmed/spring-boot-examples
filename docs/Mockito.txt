- Mockito requires you to either use only raw values or only matchers when stubbing a method call.

for example:
Invalid:
        TimeService timeService = new TimeService(restTemplate);
        when(restTemplate.getForEntity(anyString(), Map.class)).thenReturn(responseEntity);
Invalid use of argument matchers! 2 matchers expected, 1 recorded

Valid:
        TimeService timeService = new TimeService(restTemplate);
        when(restTemplate.getForEntity(anyString(), eq(Map.class))).thenReturn(responseEntity);
